## 左中右布局效果
>1. 方法一
>2. 前两个div脱离了文档流，第三个div宽度自适应
````javascript
.left {
    float: left;
}
.right {
    float: right;
}
.center {
    margin-left: 100px;
    margin-right: 100px;
}
````
````html
<div class="left">left</div>
<div class="right">right</div>
<div class="center">center</div>
````

>1. 方法二
````javascript
.all{
    position: relative;
}
.left{
    position: absolute;
    width: 100px;
    left: 0;
}
.right{
    position: absolute;
    width: 100px;
    right: 0;
}
.center{
    position: absolute;
    left: 100px;
    right: 100px;
}
````
````html
<div class="all">
    <div class="left">left</div>
    <div class="center">center</div>
    <div class="right">right</div>
</div>
````
