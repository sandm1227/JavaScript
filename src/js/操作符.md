## 操作符
### 一元操作符
````js
var num = 2;
var num2 = ++num + 2
console.log(num2) // 5
console.log(num) // 3

var num = 2;
var num2 = num-- + 2
console.log(num2) // 4
console.log(num) // 1
````
### 位操作符
> 二进制格式表示，32为
> 负数：用二进制补码表示。
>    >求二进制补码：①求数值的绝对值的二进制码
>    > ② 就二进制码的反码，再加1
````js
var num = -18
num.toString(2) // "-10010"
````
* 按位非(~)
> 按位非操作的本质：操作数的负值减1
````js
var num = 25
var num2 = ~num
console.log(num2) // -26
````
